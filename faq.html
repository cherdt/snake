Snake FAQ

What are the keyboard controls?
Left:
Left Arrow or j
Right:
Right Arrow or k
Start Game/Next Level
Up Arrow or n
Why is the board so small?
I wanted it to be playable on a smartphone.
Are there only 3 levels?
At this time, yes. I'm planning to create a level editor so that anyone can create levels.
Why?
I used to play SnakeByte on the Apple ][+ in the early 1980s. I thought it would be fun to re-create it in Javascript.
